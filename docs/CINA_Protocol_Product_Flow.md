# CINA 协议产品流程文档

## 1. 概述

CINA 协议是一个先进的去中心化稳定币协议，旨在发行与美元 1:1 锚定的稳定币 FxUSD。用户可以通过超额抵押接受的加密资产（如 WETH）来铸造（借出）FxUSD。

该协议的核心特点是其高效的清算机制和强大的锚定稳定性模块，使其能够稳健地扩展以服务大量用户。

本文档旨在详细描述协议中不同角色的工作流程和核心产品功能。

## 2. 参与角色

协议主要有四类核心参与者：

*   **借款人/头寸管理者 (Borrower):** 存入抵押品以铸造 FxUSD 的用户。他们是协议的核心使用者。
*   **FxUSD 持有者 (Holder):** 将 FxUSD 用于支付、交易或参与其他 DeFi 协议的用户。
*   **清算人 (Liquidator):** 监控协议中不健康的头寸（抵押不足），并触发清算以获取收益的参与者，是维持协议偿付能力的关键角色。
*   **套利者 (Arbitrageur):** 利用 FxUSD 在二级市场（如 Curve）与协议内部价格之间的价差进行套利的用户，是维持 FxUSD 价格锚定的重要外部力量。

## 3. 核心产品流程

### 3.1. 铸造 FxUSD (开立头寸)

这是用户与协议交互的起点。

1.  **选择抵押品:** 用户选择协议接受的一种资产（例如 WETH）作为抵押品。
2.  **存入抵押品:** 用户将选定数量的抵押品存入协议对应的资金池（Pool）。
3.  **借出/铸造 FxUSD:** 基于存入的抵押品价值和该资产的抵押率，用户可以借出一定数量的 FxUSD。例如，如果 WETH 的抵押率为 80%，存入价值 1000 美元的 WETH，用户最多可以借出 800 FxUSD。
4.  **获得头寸 NFT:** 协议会为用户创建一个代表其债务头寸的 NFT。这个 NFT 是用户对其头寸所有权的唯一凭证，可以进行转移。

**用户界面 (UI) 交互:**
*   一个清晰的仪表盘，允许用户选择抵押品类型。
*   输入框供用户输入希望存入的抵押品数量或希望借出的 FxUSD 数量。
*   实时显示头寸的健康度（抵押率/债务比率）、清算价格等关键信息。
*   用户通过钱包确认交易，完成头寸开立。

### 3.2. 管理现有头寸

用户可以随时调整其头寸以应对市场变化或管理风险。

*   **增加抵押品:** 用户可以向现有头寸添加更多抵押品，以提高其健康度，降低清算风险。
*   **减少抵押品:** 如果头寸非常健康（远高于最低抵押要求），用户可以取出部分抵押品。
*   **偿还 FxUSD:** 用户可以随时偿还部分或全部 FxUSD 债务。偿还后，其债务减少。
*   **提取全部抵押品:** 当用户完全偿还所有 FxUSD 债务后，他们可以取回所有抵押品，关闭头寸。

**UI 交互:**
*   在头寸管理页面，提供“增加抵押品”、“提取抵押品”、“偿还债务”等清晰的按钮。
*   交互式滑块或输入框，允许用户精确控制操作数量。
*   实时模拟交易后的头寸状态变化。

### 3.3. 头寸清算

清算是协议的自动风险控制机制，确保协议始终具有偿付能力。

1.  **触发条件:** 当抵押品的市场价格下跌，导致某个头寸的债务价值与抵押品价值之比超过了清算阈值时，该头寸即变为可被清算状态。
2.  **清算人介入:** 任何人（通常是运行监控机器人的清算人）都可以触发对不健康头寸的清算。
3.  **清算过程:**
    *   清算人代为偿还头寸的部分或全部 FxUSD 债务。
    *   作为回报，清算人以折扣价获得该头寸中的部分抵押品。这个折扣就是对清算人维护协议健康的激励。
4.  **协议的高效实现:** CINA 协议在技术上对清算流程进行了优化。它将健康状况相似的头寸聚合在同一个“刻度（Tick）”上。清算发生时，是直接针对整个“刻度”进行，而非单个头寸，极大地提高了效率。

### 3.4. FxUSD 锚定稳定机制

协议通过一个名为 `PegKeeper`（锚定守护者）的智能合约来主动维持 FxUSD 与美元的锚定。

1.  **价格监控:** `PegKeeper` 持续监控 FxUSD 在外部关键流动性池（如 Curve 上的 FxUSD/USDC 池）中的交易价格。
2.  **脱钩应对策略 (价格 < $1):**
    *   **暂停借款:** 如果 FxUSD 价格低于阈值（例如 $0.995），`PegKeeper` 会暂停协议新的 FxUSD 借款功能。这可以防止市场上的 FxUSD 供应量进一步增加，从而缓解抛压。
    *   **启用套利:** 同时，协议允许用户以折扣价（例如，用 1 FxUSD 赎回价值 $0.998 的抵押品）从协议中赎回抵押品。套利者会抓住这个机会：在市场上用低于 $1 的价格买入 FxUSD，然后在协议中以接近 $1 的价格赎回资产，从而获利。这个购买行为会推高 FxUSD 的市场价格，使其回归锚定。
3.  **脱钩应对策略 (价格 > $1):**
    *   如果 FxUSD 价格高于阈值（例如 $1.005），意味着市场需求旺盛。协议会确保借款功能是开启的，鼓励用户生成更多的 FxUSD 来满足市场需求，从而将价格压回 $1。

这个主动、自动化的机制是 FxUSD 能够维持其价值稳定的核心保障。
