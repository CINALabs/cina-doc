# CINA Trade：产品手册

## 1. 简介
CINA Trade 是专为 CINA 生态系统设计的高性能去中心化交易界面。它为用户提供先进的交易功能，包括杠杆多/空仓位、闪电贷以及实时的仓位监控。

## 2. 核心功能
- **杠杆交易**: 利用集成的闪电贷机制，轻松开启高达 5 倍杠杆的仓位。
- **闪电开/平仓**: 一键执行复杂的多个交易步骤（借贷、交换、铸造）。
- **钱包集成**: 通过 RainbowKit 支持 MetaMask、Coinbase Wallet 和 WalletConnect 等主流钱包。
- **实时分析**: 实时跟踪您的仓位健康度、清算价格和损益 (PnL)。

## 3. 如何使用
### 3.1 连接钱包
1. 点击右上角的 **Connect Wallet** 按钮。
2. 选择您喜欢的钱包，并在 Sepolia 测试网上授权连接。

### 3.2 开启仓位
1. 导航到 **Trade** 页面。
2. 选择您的抵押品代币（例如 `wstETH`）。
3. 输入您希望存入的抵押品数量。
4. 使用杠杆滑块选择所需的杠杆倍数（1x - 5x）。
5. 检查“预期输出 (Expected Output)”和“滑点 (Slippage)”设置。
6. 点击 **Open Position** 并在钱包中确认交易。

### 3.3 管理仓位
- 在 **Dashboard** 中查看所有活跃仓位。
- 监控 **健康因子 (Health Factor)** 以避免被清算。
- 使用 **Close** 按钮，利用闪电平仓机制立即退出仓位。

## 4. 安全说明
- 始终确保有足够的 ETH 支付 gas 费用。
- 注意健康因子；如果低于 1.0，您的仓位可能会被清算。
- 在高波动期间检查滑点设置，以确保交易能够成功执行。
